GraphW

The members in weighted graph is same as the graph, except some differences between the adjancency list representation:

Type Element in Adjancency List
T.Graph [v, [u1, u2, u3, ...]]
T.GraphW [v, [(u1, w1), (u2, w2), (u3, w3), ...]]
